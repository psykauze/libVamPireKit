# libVamPireKit
Toolkit for extracting Vampire Bloodlines .vpk

## HowTo
For now, this software need "absolutely" a Vampire Bloodlines VPK file as argument.
> vpktool pack001.vpk

You need also make an "Activision" dir in the /tmp folder
